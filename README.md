# NUST Bank Intelligent Customer Assistant

RAG-based LLM prototype for NUST Bank customer support.  
**Model:** Microsoft Phi-3.5 Mini (3.8B) | **Embeddings:** all-MiniLM-L6-v2 | **Vector Store:** FAISS

## Project Structure

```
├── data/
│   ├── NUST Bank-Product-Knowledge.xlsx   # Source product data
│   ├── faq.json                            # Source FAQ data
│   ├── qa_pairs.json                       # Generated by data_pipeline.py
│   └── chunks.json                         # Generated by data_pipeline.py
├── src/
│   ├── data_pipeline.py       # Parse XLSX + FAQ → Q&A chunks
│   ├── retriever.py           # Build FAISS index
│   ├── rag_pipeline.py        # Interactive RAG chat CLI
│   ├── download_model.py      # Download Phi-3.5 weights to local cache
│   └── app.py                 # FastAPI /chat endpoint
├── models/phi3-lora-adapter/  # LoRA adapter weights (not in repo — see Step 3)
├── notebooks/colab_train.ipynb
└── requirements.txt
```

## Setup

```bash
pip install -r requirements.txt
```

## Usage

### 1 — Data Pipeline
```bash
python src/data_pipeline.py
```
Parses `NUST Bank-Product-Knowledge.xlsx` + `faq.json` → saves `data/chunks.json` and `data/qa_pairs.json`.

### 2 — Build FAISS Index
```bash
python src/retriever.py
```
Embeds chunks with all-MiniLM-L6-v2 and builds the cosine similarity index.

### 3 — Fine-tune on Google Colab
Open `notebooks/colab_train.ipynb` in Google Colab (T4 GPU runtime).  
Upload `data/qa_pairs.json` when prompted. After training, download the adapter zip.

Extract locally:
```powershell
Expand-Archive phi3-lora-adapter.zip -DestinationPath models\phi3-lora-adapter
```

### 4 — Download Base Model
```bash
python src/download_model.py
```
Downloads Phi-3.5-mini-instruct (~7.6 GB) to the HuggingFace cache. Resumes automatically if interrupted.

### 5 — Run RAG Chat
```bash
python src/rag_pipeline.py
```
Interactive terminal chat. Out-of-domain queries (score < 0.35) are refused automatically.

### 6 — FastAPI Backend (optional)
```bash
cd src && uvicorn app:app --host 0.0.0.0 --port 8000
```
POST `/chat` with `{"query": "..."}` → `{"answer": "..."}`.

## Pipeline Flow

```
User Query
  → Embed (all-MiniLM-L6-v2)
  → FAISS search (cosine, top-5)
  → OOD guard (score < 0.35 → polite refusal)
  → Prompt: [system | retrieved context | question]
  → Generate (Phi-3.5 Mini + LoRA adapter)
  → Answer
```

## Stack

| Component   | Choice                              |
|-------------|-------------------------------------|
| LLM         | Phi-3.5-mini-instruct (3.8B)        |
| Fine-tuning | QLoRA (4-bit, r=8, alpha=16)        |
| Embeddings  | all-MiniLM-L6-v2                    |
| Vector DB   | FAISS IndexFlatIP (cosine)          |
| Backend     | FastAPI                             |
| Training    | Google Colab T4 GPU                 |
